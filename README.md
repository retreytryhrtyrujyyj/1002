# 图书推荐系统

## 项目简介
图书推荐系统包含：1.图书推荐模块 2.用户模块 3.图书分类模块 4.图书收藏模块；采用了目前最流行的前后端分离的开发模式，前端使用Vue 后端使用Springboot（（SpringMVC Spring Mybatis））框架 数据库采用Mysql。

## 技术介绍
Java;SpringBoot;Vue;Mysql

## 开发工具
1.IntelliJ IDEA或者Myeclipse 3.Navicat 3.Mysql 4.JDK8
